# ТЗ: Уточнение использования Системного Промпта в README

## 1. Общее описание
Задача: Устранить неоднозначность в README относительно использования файла `00_agent_development.md` в разных средах (Cursor vs Antigravity).
Цель: Явно указать, что в Antigravity глобальные принципы загружаются автоматически через `.gemini/GEMINI.md`, и ручная конкатенация не требуется. Добавить сравнительную таблицу.
